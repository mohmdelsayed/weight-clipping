# Weight-Clipping


## Installation:
#### 1. You need to have environemnt with python 3.11:
``` sh
conda create --name torch python==3.11
conda activate torch
```
#### 2. Install Dependencies:
```sh
python -m pip install --upgrade pip
pip install .
```
